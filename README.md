Calculator GUI CustomTkinter

Description
Calculator GUI CustomTkinter is a beginner-friendly GUI calculator built with Python using the CustomTkinter library. It provides a modern, clean interface with rounded buttons, a right-aligned display, and a black-and-white theme. This project is designed for beginners to understand GUI development in Python while creating a fully functional calculator.

Features

Basic arithmetic operations: addition (+), subtraction (-), multiplication (*), division (/)

Clear (C) button to reset the display

Equals (=) button to calculate the result

Rounded buttons for a modern look

Black-and-white color scheme for simplicity and readability

Responsive layout – buttons fill the window evenly

Right-aligned display for numbers and results

File Structure
Calculator_GUI_CustomTkinter/
├─ main.py # Main Python file containing the GUI calculator code
└─ README.txt # Project description and instructions

Installation

Make sure Python 3.6 or higher is installed on your system.

Install the required library customtkinter using pip:
python -m pip install customtkinter

How to Run

Open a terminal or command prompt.

Navigate to the project folder:
cd "path_to_your_project/Calculator_GUI_CustomTkinter"

Run the main script:
python main.py

The calculator window will appear. Click buttons to perform calculations.
